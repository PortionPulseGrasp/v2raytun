# v2RayTun — единое управление конфигурациями Xray Core

**v2RayTun** — универсальный кроссплатформенный клиент для работы с пользовательскими профилями подключений, созданными на основе **Xray Core**. Приложение доступно для **Android**, **Android TV**, **iOS**, **macOS**, **Windows** и **Linux**, объединяя инструменты для создания, импорта, изменения и хранения конфигурационных данных в единой среде управления.

Клиент поддерживает различные сценарии работы с профилями: загрузку готовых конфигурационных файлов, добавление настроек через ссылки подписок, ручное формирование параметров и редактирование ранее сохранённых записей. Все добавленные профили находятся в централизованном хранилище, благодаря чему пользователи могут удобно группировать конфигурации, контролировать актуальность данных и быстро переключаться между необходимыми вариантами подключения.

За счёт общей архитектуры **v2RayTun** обеспечивает единый принцип взаимодействия с профилями на всех совместимых устройствах — от мобильных платформ и Android TV до настольных операционных систем.

> Репозиторий **v2RayTun** содержит исходный код приложения, релизные версии, технические материалы, документацию, сведения о поддерживаемых платформах, историю обновлений и информацию о дальнейшем развитии проекта.

## Управление профилями подключений

> [!WARNING]
> **v2RayTun** является клиентским приложением для **Xray Core** и не предоставляет собственные серверные ресурсы, VPN-сервисы, прокси-инфраструктуру, подписочные решения или готовые конфигурации подключения.

Для использования приложения необходимо добавить собственные параметры одним из доступных способов:

* импортировать конфигурационный файл;
* подключить профиль через ссылку подписки;
* создать настройки вручную.

После добавления профилей пользователь может изменять параметры конфигураций, управлять сохранённым списком настроек и выбирать активный вариант подключения.

## Технологии и поддерживаемые платформы

Архитектура **v2RayTun** разрабатывается с учётом особенностей каждой операционной системы. Использование отдельных технологических решений для разных платформ позволяет обеспечить стабильную работу, рациональное использование ресурсов и корректную интеграцию с возможностями устройств.

Некоторые возможности могут добавляться постепенно из-за различий системных API и технических ограничений отдельных платформ.

| Платформа            | Технологии     | Особенности                                                   |
| -------------------- | -------------- | ------------------------------------------------------------- |
| Android / Android TV | Kotlin         | Адаптация для мобильных устройств и телевизионных платформ    |
| Windows / Linux      | Flutter + Dart | Кроссплатформенная реализация с настройками под разные сборки |
| iOS                  | Swift          | Нативное приложение для экосистемы Apple                      |
| macOS Apple Silicon  | Swift          | Оптимизация для устройств с ARM-чипами Apple                  |
| macOS Intel          | Swift          | Совместимость с компьютерами Mac на базе Intel                |
